# 전처리 또는 선행처리 단계를 설명하라
컴파일 전에 지시문을 처리하여 필요한 파일을 포함하고 코드를 미리 처리하는 단계이다.

# 객체 cin, cout은 어디에 선언되어 있는가?
cin과 cout은 iostream 헤더파일에 선언되어 있으며 입출력을 처리하기 위한 라이브러리이다.

# 프로그램을 링크할 때 이름 충돌이 발생하는 경우를 설명하라
서로 다른 파일에 같은 이름의 함수나 변수가 있을 때 이름 충돌이 발생한다.

# 스트림이란 무엇인가?
스트림은 프로그램과 입출력 장치 사이에서 데이터가 흐르는 통로이다.

# 표준 출력 스트림을 정의할 때 사용하는 C언어 자료형은?
C언어에서는 FILE 자료형을 사용한다.

# 표준 출력 스트림을 정의할 때 사용하는 C++언어 자료형은?
C++에서는 ostream 자료형을 사용한다.
