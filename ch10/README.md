#  배열명의 의미를 설명하라.
배열명은 배열의 첫 번째 원소의 주소이다.
# 객체 포인터의 의미를 설명하라.
객체 포인터는 객체의 주소를 저장하는 변수이다.
# 포인터를 이용하여 배열 각 원소의 주소를 구하는 방법을 설명하라
포인터에 인덱스를 더한 (p + i)로 구한다.
# 포인터를 이용하여 배열 각 원소의 값을 구하는 방법을 설명하라
역참조 *(p + i) 또는 p[i]로 구한다.
