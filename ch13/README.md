# 실습과제 1

# string 객체는 문자열 입력 받을 때 몇 자를 입력할지 물어 보지않고 알아서 크기를 자동 조절한다. 동작 원리를 검색해보고 설명하라 -> 앞으로, C++에서는 char배열을 사용하지 말고 string 클래스를 사용할 것
string 클래스는 내부적으로 동적 메모리 할당을 사용하여 문자열 크기에 맞게 메모리를 자동으로 늘리거나 줄인다.
# 문자열을 입력할 때 마지막에 입력하는 엔터키와 getline 함수의 마지막 인자에 전달하는 구분문자와 차이를 설명하라
엔터키(\n)는 사용자의 입력을 끝내는 역할을 하고,getline 함수의 구분문자는 문자열을 어디까지 읽을지 결정하는 기준이다.



# 실습과제 2

# 소스코드 설명
```
#include <iostream>
```
표준 입출력을 사용하기 위해 iostream 헤더파일을 포함한다.
```
#include <string>
```
문자열 처리를 위해 string 헤더파일을 포함한다.
```
using namespace std;
```
std 네임스페이스를 사용하여 std:: 없이 cin, cout, string 등을 사용한다.
```
int main()
```
프로그램이 시작되는 메인 함수이다.
```
string s, ss;
```
입력받은 문자열을 저장할 s와 이어붙인 결과를 저장할 ss를 선언한다.
```
int num = 1;
```
문자열 앞에 붙일 번호를 저장하는 변수이며 1로 초기화한다.
```
while (true)
```
무한 반복을 수행한다.
```
cout << "문자열 입력: ";
```
사용자에게 문자열 입력을 안내한다.
```
getline(cin, s, '\n');
```
한 줄의 문자열을 입력받아 s에 저장한다.
```
if (s == "quit") break;
```
입력값이 "quit"이면 반복문을 종료한다.
```
ss += '<' + to_string(num) + '>' + s;
```
번호와 함께 문자열을 이어붙여 ss에 저장한다.
```
cout << "이어진 문자열: " << ss << endl;
```
현재까지 이어진 문자열을 출력한다.
```
num++;
```
다음 번호를 위해 값을 증가시킨다.
```
return 0;
```
프로그램을 정상적으로 종료한다.

# 실습과제 3

# 소스코드 설명
```
#include <iostream>
```
표준 입출력을 사용하기 위해 iostream 헤더파일을 포함한다.
```
#include <string>
```
문자열 처리를 위해 string 헤더파일을 포함한다.
```
using namespace std;
```
std 네임스페이스를 사용하여 std:: 없이 cin, cout, string 등을 사용한다.
```
int main()
```
프로그램이 시작되는 메인 함수이다.
```
string s;
```
입력받은 문자열을 저장할 변수이다.
```
int cnt = 0;
```
문자 'a'의 개수를 세기 위한 변수이며 0으로 초기화한다.
```
cout << "문자열 입력: ";
```
사용자에게 문자열 입력을 안내한다.
```
getline(cin, s, '\n');
```
한 줄의 문자열을 입력받아 s에 저장한다.
```
for (int i = 0; i < s.length(); i++)
```
문자열의 길이만큼 반복하면서 각 문자를 확인한다.
```
if (s.at(i) == 'a')
```
현재 문자가 'a'인지 확인한다.
```
cnt++;
```
문자가 'a'일 경우 개수를 1 증가시킨다.
```
cout << "문자 a는 " << cnt << "개 있습니다.";
```
문자열에 포함된 'a'의 총 개수를 출력한다.
```
return 0;
```
프로그램을 정상적으로 종료한다.
