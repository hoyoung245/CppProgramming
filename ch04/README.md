# 실습과제1

# 헤더파일과 목적파일에 들어 있는 내용이 무엇인지 각각 설명하시오.
헤더파일: 함수의 선언, 클래스 정의 등이 들어 있는 파일로 프로그램에서 사용할 기능을 알려준다.

목적파일: 함수의 실제 구현(정의)이 들어 있는 파일이다.

# 함수의 선언, 정의, 호출은 각각 어디에 존재하는가?
선언: 함수의 이름과 형태를 알리는 것 (헤더파일에 존재)

정의: 함수의 실제 동작을 작성한 것 (소스파일에 존재)

호출: 프로그램에서 함수를 사용하는 것 (main 함수 등에서 사용)

# <iostream> 헤더파일에 cin, cout은 extern 변수로 선언되어 있다.extern 선언과 변수선언의 차이를 설명하라. 그리고, 그럼 cin, cout의 변수선언은 어디에 있는가?
extern 선언: 다른 파일에 이미 정의된 변수를 사용하겠다고 알리는 선언

변수 선언: 실제로 메모리를 할당하여 변수를 만드는 것

cin, cout의 변수 선언 위치: C++ 표준 라이브러리 내부에 정의되어 있다.

# 라이브러리 함수나 객체를 사용하기 전에 헤더파일을 include하는 이유를 설명하라.
컴파일러에게 함수와 객체의 선언을 알려주어 오류 없이 사용할 수 있도록 하기 위해서이다.
