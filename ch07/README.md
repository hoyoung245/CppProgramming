# 실습과제 1

#  객체의 생성과 소멸의 의미를 설명하라
객체의 생성은 메모리에 객체가 만들어지는 것이고, 소멸은 객체가 사용이 끝나 메모리에서 사라지는 것이다.
# 지역변수(지역객체), 전역변수(전역객체)의 차이를 조사하라, 선언위치, 접근범위, 생성시점, 소멸시점, 초기화 여부 등의 관점에서 비교하라
선언 위치
지역객체: 함수 안에서 선언
전역객체: 함수 밖에서 선언
접근 범위
지역객체: 해당 함수 안에서만 사용
전역객체: 프로그램 전체에서 사용 가능
생성 시점
지역객체: 함수가 실행될 때 생성
전역객체: 프로그램 시작 시 생성
소멸 시점
지역객체: 함수가 끝나면 소멸
전역객체: 프로그램 종료 시 소멸
초기화 여부
지역객체: 자동 초기화되지 않음
전역객체: 자동으로 0으로 초기화됨
# 프로그램 실행 후 main함수가 실행되기 전에 어떤 일이 처리되는지 조사해보라
전역 변수와 전역 객체가 먼저 생성되고 초기화된다.
# main함수가 종료 후 프로그램이 종료하기 전에 어떤 일이 처리되는지 조사하라
전역 객체가 소멸되고 프로그램이 종료된다.
