# 소스코드 설명
```
#include <iostream>
```
표준 입출력을 사용하기 위해 iostream 헤더파일을 포함한다.
```
using namespace std;
```
std 네임스페이스를 사용하여 std:: 없이 cin, cout 등을 사용한다.
```
class Rectangle
```
사각형을 표현하는 Rectangle 클래스를 선언한다.
```
private:
```
외부에서 직접 접근할 수 없는 private 영역을 선언한다.
```
int width;
```
사각형의 가로 길이를 저장하는 변수이다.
```
int height;
```
사각형의 세로 길이를 저장하는 변수이다.
```
public:
```
외부에서 사용할 수 있는 멤버를 선언하는 접근 지정자이다.
```
Rectangle()
```
기본 생성자로 width와 height를 1로 초기화한다.
```
width = 1;
```
가로 길이를 1로 설정한다.
```
height = 1;
```
세로 길이를 1로 설정한다.
```
Rectangle(int w)
```
매개변수 1개를 받는 생성자로 width를 설정하고 height는 1로 초기화한다.
```
width = w;
```
가로 길이를 w로 설정한다.
```
height = 1;
```
세로 길이를 기본값 1로 설정한다.
```
Rectangle(int w, int h)
```
매개변수 2개를 받는 생성자로 width와 height를 모두 초기화한다.
```
width = w;
```
가로 길이를 w로 설정한다.
```
height = h;
```
세로 길이를 h로 설정한다.
```
void show()
```
사각형의 정보를 출력하는 함수이다.
```
cout << "사각형 폭은 " << width << " 높이는 " << height << endl;
```
현재 객체의 가로와 세로 값을 출력한다.
```
int main()
```
프로그램이 시작되는 메인 함수이다.
```
Rectangle rect0;
```
기본 생성자를 이용하여 객체 rect0을 생성한다.
```
rect0.show();
```
rect0의 값을 출력한다.
```
Rectangle rect1(10);
```
가로만 10으로 설정된 객체 rect1을 생성한다.
```
rect1.show();
```
rect1의 값을 출력한다.
```
Rectangle rect2(10, 20);
```
가로 10, 세로 20으로 설정된 객체 rect2를 생성한다.
```
rect2.show();
```
rect2의 값을 출력한다.
```
return 0;
```
프로그램을 정상적으로 종료한다.

# 실습과제 4

# 소스코드 설명
```
#include <iostream>
```
표준 입출력을 사용하기 위해 iostream 헤더파일을 포함한다.
```
using namespace std;
```
std 네임스페이스를 사용하여 std:: 없이 cin, cout 등을 사용한다.
```
class Point3D
```
3차원 좌표를 표현하는 Point3D 클래스를 선언한다.
```
private:
```
외부에서 직접 접근할 수 없는 private 영역을 선언한다.
```
int x, y, z;
```
3차원 좌표의 x, y, z 값을 저장하는 정수형 변수이다.
```
public:
```
외부에서 사용할 수 있는 멤버를 선언하는 접근 지정자이다.
```
Point3D()
```
기본 생성자로 모든 좌표를 0으로 초기화한다.
```
x = 0;
```
x좌표를 0으로 설정한다.
```
y = 0;
```
y좌표를 0으로 설정한다.
```
z = 0;
```
z좌표를 0으로 설정한다.
```
Point3D(int a)
```
x값만 입력받고 y, z는 0으로 초기화하는 생성자이다.
```
x = a;
```
x좌표를 a로 설정한다.
```
y = 0;
```
y좌표를 0으로 설정한다.
```
z = 0;
```
z좌표를 0으로 설정한다.
```
Point3D(int a, int b)
```
x, y값을 입력받고 z는 0으로 초기화하는 생성자이다.
```
x = a;
```
x좌표를 a로 설정한다.
```
y = b;
```
y좌표를 b로 설정한다.
```
z = 0;
```
z좌표를 0으로 설정한다.
```
Point3D(int a, int b, int c)
```
x, y, z 모두 입력받아 초기화하는 생성자이다.
```
x = a;
```
x좌표를 a로 설정한다.
```
y = b;
```
y좌표를 b로 설정한다.
```
z = c;
```
z좌표를 c로 설정한다.
```
void show()
```
3차원 좌표를 출력하는 함수이다.
```
cout << "3차원 점의 좌표는 (" << x << "," << y << "," << z << ")" << endl;
```
현재 객체의 좌표 값을 출력한다.
```
int main()
```
프로그램이 시작되는 메인 함수이다.
```
Point3D p0;
```
기본 생성자를 이용하여 객체 p0을 생성한다.
```
p0.show();
```
p0의 좌표를 출력한다.
```
Point3D p1(1);
```
x만 1로 설정된 객체 p1을 생성한다.
```
p1.show();
```
p1의 좌표를 출력한다.
```
Point3D p2(1, 2);
```
x, y가 설정된 객체 p2를 생성한다.
```
p2.show();
```
p2의 좌표를 출력한다.
```
Point3D p3(1, 2, 3);
```
x, y, z가 모두 설정된 객체 p3을 생성한다.
```
p3.show();
```
p3의 좌표를 출력한다.
```
return 0;
```
프로그램을 정상적으로 종료한다.
