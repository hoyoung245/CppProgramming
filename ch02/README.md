# IDE란 무엇인가?

IDE는 코드를 작성, 편집, 컴파일, 실행, 디버깅까지 한 곳에서 할 수 있도록 통합된 개발용 소프트웨어이다.

# IDE의 종류에 대하여 조사하시오.

IDE의 종류에는 Visual Studio, Eclipse, Code::Blocks, PyCharm, IntelliJ IDEA 등이 있으며, 각각 지원하는 언어와 기능이 다르다.

# IDE에 포함된 기능은 무엇인가?

IDE에 포함된 기능은 코드 편집, 컴파일/실행, 디버깅, 빌드 관리, 버전 관리 연동 등이다.

# 소스코드 설명
```
#include <iostream>
```
cin, cout 등 라이브러리 객체의 선언을 포함하고 있는 헤더 파일 iostream을 포함함

```
int main(void)
```
프로그램의 실행이 시작되는 메인 함수 시작

```
std::cout << "Hello world!!";
```
화면에 "Hello world!!"를 출력함.std::cout은 화면에 출력해주는 객체, << 는 삽입 연산자 함수임

```
std::cout << std::endl;
```
화면에 줄바꿈 문자(std::endl)를 출력하고 출력 버퍼를 비움

```
return 0;
```
0을 반환하고 메인 함수를 정상적으로 종료함
